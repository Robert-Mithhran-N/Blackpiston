# BlackPiston Garage

Premium motorcycle gear and workshop experience built with Vite, React, TypeScript, Tailwind CSS, and shadcn/ui.

## Getting Started

```sh
npm install
npm run dev
```

## Scripts
- `npm run dev` – start the dev server
- `npm run build` – production build
- `npm run preview` – preview the production build locally
- `npm run lint` – lint the project

## Tech Stack
- React 18 + Vite
- TypeScript
- Tailwind CSS + shadcn/ui
- React Router

## Notes
- Assets live in `public/` and `src/assets/`.
- Update the metadata in `index.html` if you change branding, OG images, or social previews.
